Solution to question 4: JavaScript HTML DOM

Qual a diferença entre as propriedades textContent e innerHTML dos elementos HTML? Cite exemplos.

As propriedades textContent e innerHTML são usadas para acessar ou alterar o conteúdo de um elemento HTML, mas existem algumas diferenças importantes entre elas.

A propriedade textContent retorna ou define apenas o texto dentro do elemento, sem qualquer marcação HTML. Já a a propriedade innerHTML retorna ou define o conteúdo completo do elemento, incluindo qualquer marcação HTML dentro dele.
